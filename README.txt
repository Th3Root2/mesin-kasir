Ini merupakan projek tim,yang beranggotakan ;
1. Fatih
2. Radit
3. Kiki
4. Nabila

projek ini bertemakan mesin kasir, yang dimana role admin dapat melakukan CRUD. 
Create yang dimana admin dapat menambahkan member baru.
Read untuk admin bisa membaca data member ataupun admin.
Update yang dimana memungkinkan admin merubah informasi member.
Delete, membuat admin dapat menghapus member

Note : Fitur CRUD tidak hanya untuk admin yg mengelola member, melainkan untuk produk juga admin dapat mengelolanya.
